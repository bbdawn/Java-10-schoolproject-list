**school project list**

- 요구사항
학교 구성원 정보를 관리하는 시스템을 구축하고자한다.

학교 구성원 정보는 학생(Student), 교사(Teacher),직원(Employee)로 구성된다.

학생은 전화번호(tel), 이름(name), 주소(address), 학번(stdID)으로 구성된다
교사는 전화번호(tel), 이름(name), 주소(address), 과목(subject)으로 구성된다
직원은 전화번호(tel), 이름(name), 주소(address), 부서(department)로 구성된다


- 주요 기능
1. 구성원 정보를 등록
	학교 구성원 정보를 등록하여 유지할 수 있다
	등록시 등록순서를 보장하고 전화번호는 구성원당 유일해야하므로 중복된 전화번호 구성원일경우 등록시키지 않는다. 
	(중복시 등록 불가 메세지 : 01012341234 tel이 중복되므로 등록 불가합니다)


2. 전체 구성원 리스트를 화면에 출력 (구성원의 상세 정보가 모두 출력)

3. 전화번호(tel)를 이용한 구성원 정보를 검색한다
	검색할 구성원 정보의 전화번호를 전달하면 해당하는 구성원(학생 or 교사 or 직원)정보를 검색해 반환한다
	만약 전화번호에 해당하는 구성원 정보가 없으면 null을 반환
	
4. 전화번호(tel) 를이용한 구성원 정보를 삭제
   삭제할 구성원 정보의 전화번호를 전달하면 리스트에서 해당 구성원 정보를 삭제한다
   만약 전화번호에 해당하는 구성원 정보가 없다면 아래와 메세지를 출력한다
   ( 출력 메세지 : 01012341234 tel에 해당하는 구성원 정보가 없어 삭제 불가)
